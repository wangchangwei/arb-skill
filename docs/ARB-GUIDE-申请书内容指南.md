> 本模板帮助用户**填写内容**，格式按当地仲裁委要求来。

## 二、仲裁请求（核心部分）

### 请求事项（逐条列明）

| 序号 | 请求内容 | 金额（元） | 计算依据 |
|------|---------|-----------|---------|
| 1 | 支付加班费 | XX,XXX.XX | 见附件：加班费计算明细 |
| 2 | 支付违法解除赔偿金 | XX,XXX.XX | 工作X年×月薪×2N |
| 3 | 支付代通知金 | XX,XXX.XX | 1个月工资 |
| 4 | 支付未休年假折算 | X,XXX.XX | X天×日薪×3倍 |
| 5 | 支付未签合同双倍工资差额 | XX,XXX.XX | X个月×月薪 |
| 6 | ... | ... | ... |

**填写方法**：
- 金额精确到分
- 计算依据要写"见附件"，不现场展开
- 每项请求对应一个计算明细（附件）

## 三、事实与理由（核心叙述）

### 模板框架

```
申请人在被申请人处工作，工作岗位为XXXX，月薪XXXX元，合同期限XXXX。

一、关于加班费
申请人入职以来，长期存在加班情况。2026年X月至2026年X月期间，
申请人累计加班XXX小时，被申请人未支付加班工资。
（附：考勤记录、聊天记录截图等证据）

二、关于违法解除劳动合同
XXXX年X月X日，被申请人无合法理由单方解除与申请人的劳动合同，
未提前30日通知，未支付经济补偿金。
（附：解除通知截图、聊天记录等证据）

三、关于未休年假
申请人在职期间每年应享受年假X天，实际未休X天，被申请人未支付折算工资。
（附：考勤记录或请假记录）

综上所述，被申请人的行为严重侵害了申请人的合法权益。
申请人依据《劳动合同法》第XX条、第XX条等规定，
提出上述仲裁请求。
```

### 填写要点

| 要素 | 要求 |
|------|------|
| 时间线 | 写清楚年月日，事件先后顺序要连贯 |
| 金额 | 写清楚计算逻辑，不写死金额（最终以证据为准） |
| 法律依据 | 写明条款编号，不要空着 |
| 语气 | 客观陈述事实，不写情绪性语言 |

## 五、常用法律条款（按情形选择）

### 加班费主张
- 《工资支付暂行规定》第13条
- 《劳动法》第44条
- 《劳动合同法》第31条

### 违法解除赔偿金
- 《劳动合同法》第47条（经济补偿）
- 《劳动合同法》第87条（2N赔偿）
- 《劳动合同法》第40条（代通知金）

### 未签合同双倍工资
- 《劳动合同法》第82条

### 未休年假
- 《职工带薪年休假条例》第5条

## 六、申请书填写自检清单

- [ ] 申请人信息填写完整、无错漏
- [ ] 仲裁请求金额与计算依据一致
- [ ] 事实描述有时间线、逻辑清晰
- [ ] 法律条款编号准确
- [ ] 证据清单与申请书对应
- [ ] 证据材料按清单顺序整理
- [ ] 劳动合同原件准备好
- [ ] 身份证复印件准备好
- [ ] 所有证据复印2份（1份提交，1份自留）
