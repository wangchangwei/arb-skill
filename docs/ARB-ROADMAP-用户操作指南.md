# 劳动仲裁 Roadmap - 用户操作指南

> 帮用户从**准备证据**到**拿到赔偿金**，每一步清晰指引。

---

## 阶段一：收集证据（工具辅助）

| 步骤 | 操作 | 工具 | 说明 |
|------|------|------|------|
| 1.1 | 收集考勤记录 | 飞书工作台导出 Excel | 导出路径：工作台 → 考勤 → 统计报表 |
| 1.2 | 收集工资流水 | 银行App导出 | 打印12个月流水，盖银行章 |
| 1.3 | 收集劳动合同 | 公司HR或自己的文件 | 电子合同/纸质合同都行 |
| 1.4 | 收集社保缴纳记录 | 社保中心官网/小程序 | 证明劳动关系和时长 |
| 1.5 | 收集聊天记录截图 | 微信/飞书 | 加班指令、辞退通知等 |
| 1.6 | 上传文件给 Agent | Claude Code | 分析证据、计算金额 |

**自检**：以上材料至少要有劳动合同 + 考勤记录 + 工资流水

---

## 阶段二：证据分析与金额计算（工具生成）

| 步骤 | 操作 | 工具 | 输出 |
|------|------|------|------|
| 2.1 | 上传考勤 Excel | `parse_attendance_file_ai` | 考勤数据分析结果 |
| 2.2 | 上传聊天截图 | Agent Vision 直接分析 | 聊天证据分析报告 |
| 2.3 | 计算权益金额 | `calculate_claims` | 各项赔偿金额明细 |
| 2.4 | 生成证据分析报告 | `analyze_evidence` | 有利/不利/缺失清单 |

**自检**：
- [ ] 加班费金额已算出
- [ ] 赔偿金类型已确定
- [ ] 证据强度评级完成
- [ ] 缺失证据已标注

---

## 阶段三：准备仲裁申请材料

| 步骤 | 操作 | 说明 |
|------|------|------|
| 3.1 | 填写仲裁申请书 | 按《申请书内容指南》填写 |
| 3.2 | 复印所有证据材料 | 每种证据复印2份 |
| 3.3 | 整理证据目录 | 按清单顺序排列 |
| 3.4 | 准备身份证原件+复印件 | 原件窗口验，复印件1份 |

**自检**：
- [ ] 申请书各项金额与工具计算一致
- [ ] 证据清单与申请书对应
- [ ] 所有材料按顺序整理完毕

---

## 阶段四：提交仲裁申请

| 步骤 | 操作 | 地点/方式 |
|------|------|-----------|
| 4.1 | 确定管辖仲裁委 | 公司注册地所在区仲裁委 |
| 4.2 | 查地址和电话 | 地图搜索「XX区劳动争议仲裁委员会」 |
| 4.3 | 窗口提交 or 邮寄 | 窗口当场拿受理回执，邮寄保留单号 |
| 4.4 | 缴纳仲裁费 | **免费**（劳动者一方） |

**⚠️ 注意时效**：从知道权益被侵害起算，**1年内**必须提交

---

## 阶段五：仲裁委处理

| 阶段 | 时间 | 你的操作 |
|------|------|---------|
| 📞 调解阶段 | 提交后5工作日内 | 仲裁员可能电话调解，可拒绝 |
| ⚖️ 开庭通知 | 受理后5工作日内 | 查短信/邮件，等待开庭通知 |
| 开庭 | 通知的日期 | 携带身份证原件、所有证据原件到场 |
| 等待裁决 | 45天内 | 审结后领取裁决书 |

---

## 阶段六：裁决后执行

| 情况 | 行动 | 期限 |
|------|------|------|
| 公司不服裁决 | 15天内向基层法院起诉 | — |
| 公司不执行裁决 | 向法院申请**强制执行** | 2年内 |
| 胜诉且公司主动履行 | 等待打款 | — |

---

## 关键时间节点

```
知道权益被侵害
    │
    ▼ 1年内
提交仲裁申请
    │
    ▼ 5工作日内
仲裁委受理
    │
    ▼ 45天内
裁决出具
    │
    ├──► 公司履行 → 结束 ✓
    │
    └──► 公司不履行
            │
            ▼ 15天内
        法院起诉（公司权利）
            或
        申请强制执行（公司不履行）
```
